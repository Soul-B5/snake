
# **游戏介绍**:

## **运行条件**
- Python 3.6+
- 在终端文件目录下运行 `python game.py`,即可启动


## **游戏规则**
- 游戏地图由n*n的格子组成的围墙构成。
- 食物在地图中随机生成5个`*`形食物。
- 吃到食物，蛇的长度增加1。    

## **操作说明**
- 蛇的初始位置在左上角,方向向右
- 蛇的运动方向由键盘控制, 按下W键向上, S键向下, A键向左, D键向右
- 按q键可退出游戏。
- 当蛇撞到障碍物, 游戏结束。
